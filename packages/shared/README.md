# @verse/shared

Shared logic for the Verse by Verse web app and future React Native app.

Recommended first migrations into this package:

- API base/config helpers
- auth/session utilities
- community normalization helpers
- notes request helpers
- shared constants and validation rules

Current shared exports:

- `getApiBase`
- `buildApiUrl`
- `getAssetUrl`
- `buildHeroStyle`
- `createApiClient`
- `createAuthApi`
- `createNotesApi`
- `formatRelativeTime`
- `normalizeCommunity`
